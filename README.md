Native Apple Wallet and Google Wallet loyalty stamp cards


 
for small merchants in the GCC and Europe.

🔗 [btaqa.io](https://btaqa.io)

## What it does
- Merchants create branded loyalty cards in minutes
- Customers add to Apple or Google Wallet with one tap
- Live pass updates on customer lock screen via APNs/FCM
- Geofencing notifications
- Full Arabic/RTL support
- No app required

## Stack
- Next.js App Router
- Supabase
- Vercel
- Capacitor (Scanner App)
- passkit-generator (Apple Wallet)
- Firebase FCM + APNs
